INSTRUCTIONS TO RUN CYPRESS TESTS

1. Install node.js
2. In local terminal of your choice run 'npm install cypress --save-dev'
3. In terminal go to project repo as PWD
4. Run cypress with 'npx cypress open' or './node_modules/.bin/cypress open' -a first time set up will happen
5. When cypress UI pops up just click a test and a new chrome window will open running the selected cypress test
6. If you run into any issues installing or launching cypress refere to cypress documentation at 'https://docs.cypress.io'
